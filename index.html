Welcome to Tunicates GitHub.

This is GitHub for Tunicates data. Latest data is easily downloaded by Git command like this.

$ git pull github.com/Tunicates/Repo/Ciona_intestinalis

If you would like to download all the latest data, just do this:

$ git pull github.com/Tunicates/Repo

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
